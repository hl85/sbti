# SBTI 人格测试

> MBTI已经过时，SBTI来了。

SBTI（Silly Big Personality Test）—— 31道灵魂拷问，26种抽象人格，精准拿捏你的精神状态。

本测试首发于 [B站@蛆肉儿串儿](https://space.bilibili.com/417038183)，本项目为 Vue 3 精致复刻版。

## ✨ 特性

- 🧠 31道灵魂拷问 + 2道隐藏触发题
- 🎭 26种抽象人格（含隐藏人格"酒鬼"）
- 📊 十五维度可视化评分
- 📸 一键导出结果为图片
- 🔗 生成分享链接，支持微信/微博/QQ/豆瓣/贴吧分享
- 📱 移动端适配
- 🎨 精致 UI，页面切换动画

## 🛠️ 技术栈

- Vue 3 + Composition API
- Vue Router 4
- Vite
- html2canvas（图片导出）

## 🚀 本地开发

```bash
npm install
npm run dev
```

## 📦 构建

```bash
npm run build
```
